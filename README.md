# Weather Application (React + Node.js)

A modern, responsive weather application built with **React**, **Vite**, and **Tailwind CSS**, featuring a **Glassmorphism UI** and dynamic background themes.

## Features
- **Real-time Weather**: Powered by Weatherstack API.
- **React Frontend**: Fast, component-based UI with Vite.
- **Dynamic Backgrounds**: Changes based on weather conditions (Sunny, Rain, Snow, etc.).
- **Glassmorphism Design**: Premium UI with Tailwind CSS.

## Prerequisites
- [Node.js](https://nodejs.org/) installed.
- API Key in `.env` file (`WEATHERSTACK_API_KEY`).

## Installation
1.  Install dependencies:
    ```bash
    npm install
    ```

## Usage
To run the application, you need to start **both** the backend and the frontend.

1.  **Start the Backend API**:
    ```bash
    npm run server
    ```
    (Runs on `http://localhost:3000`)

2.  **Start the React Frontend**:
    Open a new terminal and run:
    ```bash
    npm run dev
    ```
    (Runs on `http://localhost:5173`)

3.  Open your browser to the link shown in the terminal (usually `http://localhost:5173`).
